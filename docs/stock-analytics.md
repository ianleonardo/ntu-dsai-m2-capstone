# Stock Analytics

## Business Narrative

Build a 'free' bespoke Stock Price Monitor for a finance company. (Stretch target: Automate technical analysis, e.g. simple moving average, support & resistance levels with aid of LLM tools.)

## Data Source 
Finnhub Stock APIs [https://finnhub.io/](https://finnhub.io/)
Data Provided: Real-time stock, forex, and crypto, 30+ years of historical data, company fundamentals, and news.
API Format: REST API (JSON) and WebSockets for real-time streaming.
Free Tier: Free API key for individual developers.

## How-to guide
[https://www.youtube.com/watch?v=Nu3bEtNrmIw](https://www.youtube.com/watch?v=Nu3bEtNrmIw). 
➣ Use the Finnhub API to fetch live stock data.
➣ Process JSON data with Python.
➣ Create a dynamic web app with Streamlit.
➣ Display real-time data in a table format using Pandas.
